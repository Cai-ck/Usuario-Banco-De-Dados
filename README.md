## Sobre o Projeto
Desenvolvimento de uma aplicação backend para controle de acesso, com foco em segurança da informação. 
O sistema permite o registro de novos usuários e a validação de login, implementando uma camada de criptografia que impede a visualização de senhas em texto claro diretamente no banco de dados, unindo lógica de programação com gestão de dados relacionais.

## Bibliotecas usadas *(Por enquanto)*
- **Asyncpg**: Operações assíncronas projetada para o PostgreSQL.
- **Bcrypt**: Para hashing de senhas.
  
## Tecnologias utilizadas *(Por enquanto)*

- **Python3**: Conexão com o database, autenticação e login.
- **DBeaver**: Ferramenta Gráfica e gerenciamento do banco de dados.
- **PostgreSQL**: Criar uma banco de dados, para amazenar as informações dos usuários e seus jogos.
